# Algorithms Visualization

A modern, interactive web application to visualize various sorting and pathfinding algorithms. Built with React, TypeScript, and Vite, featuring a sleek UI powered by Tailwind CSS.

## Features

- **Search Visualization:** Interactive demonstrations of search algorithms like Binary Search and Linear Search.
- **Pathfinding Visualization:** Real-time visualization of pathfinding algorithms (Dijkstra's, A*, BFS, DFS).
- **Sorting Algorithms:** (To be fully implemented) Visualize sorting logic step-by-step.
- **Real-time Stats:** Track performance and complexity as algorithms run.
- **Audio Engine:** Interactive sound effects to represent algorithm progress.
- **Confetti & Winner Toast:** Celebration UI for algorithm completion.

## Tech Stack

- **React 19**
- **Vite**
- **TypeScript**
- **Tailwind CSS 4**
- **Lucide React** (Icons)
- **clsx & tailwind-merge** (Utility functions)

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MuhammadAmmarKhan/algorithms-visualization.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

This project is deployed to GitHub Pages. Visit [https://MuhammadAmmarKhan.github.io/algorithms-visualization/](https://MuhammadAmmarKhan.github.io/algorithms-visualization/) to see it in action.
